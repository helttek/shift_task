# Task for CFT Shift

## Dependencies

- Maven 3.8.7
- Java 21.0.8

## Libraries

- JUnit 3.8.1
- Apache Common CLI 1.11.0

## Build

from directory with pom.xml (FileContentSorter):

```bash
mvn clean install
```

## Launch

X - version (current is 0.1)

```bash
java -jar target/FileContentSorter-X.jar
```
